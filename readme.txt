This plugin allows you to automatically produce 301 redirects to the home page or another URL for deleted posts and taxonomies to avoid 404 errors.

== Description ==

Auto Redirect 404 in 301 for Trashed Posts is a plugin that allows to treat the SEO of your website.

Indeed, once an post is deleted or a taxonomy, the plugin automatically create a 301 redirect of your choice to avoid 404 errors and be penalized by Google.

By default, the old posts and taxonomies are redirected to the home page of your site.
You can change the redirect URL for every Post Type, Taxonomy or Posts/Pages

Please note that posts that are present in the trash have a 302 temporary redirect.

Translation: English, French 

== Installation ==

1. Upload the complete `auto-redirect-404` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Auto Redirect 404' under the 'Settings' tab and configure the plugin
